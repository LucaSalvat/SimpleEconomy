PROBLEM: Homepage is empty, can't click articles, nothing shows up

SOLUTION: You need to use a local web server!

--------------------------------------------------------------------------------
                         ‚ö° FASTEST FIX (30 seconds)
--------------------------------------------------------------------------------

1. Open Terminal/Command Prompt

2. Type this:
   cd economics-notes
   
3. Type this:
   python -m http.server 8000
   
   (If that doesn't work, try: python3 -m http.server 8000)

4. You should see:
   "Serving HTTP on 0.0.0.0 port 8000..."

5. Open your browser and go to:
   http://localhost:8000

6. DONE! ‚úÖ Everything should work now!

--------------------------------------------------------------------------------
                              WHY THIS WORKS
--------------------------------------------------------------------------------

Your website loads articles from a JSON file using JavaScript.

‚ùå Opening index.html directly = Browser blocks JSON loading (CORS security)
‚úÖ Using a server = Browser allows JSON loading

It's not broken - you just need to run it properly!

--------------------------------------------------------------------------------
                           OTHER EASY OPTIONS
--------------------------------------------------------------------------------

OPTION 1: VS Code Live Server
  - Install "Live Server" extension in VS Code
  - Right-click index.html ‚Üí "Open with Live Server"
  - Auto-refreshes when you edit!

OPTION 2: Node.js
  - npx http-server -p 8000
  - Then go to: http://localhost:8000

OPTION 3: PHP
  - php -S localhost:8000
  - Then go to: http://localhost:8000
