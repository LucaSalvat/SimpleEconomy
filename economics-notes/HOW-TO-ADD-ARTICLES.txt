================================================================================
                    üìù HOW TO ADD NEW ARTICLES
================================================================================

Quick guide for adding new economics notes to your website.

================================================================================
                          STEP 1: CREATE ARTICLE HTML
================================================================================

1. Go to the 'articles' folder

2. Copy 'article-template.html'

3. Rename it:
   Example: international-trade-theory.html
   
   Rules:
   - Use lowercase
   - Use hyphens (not spaces)
   - No special characters
   ‚úÖ supply-and-demand.html
   ‚ùå Supply & Demand!.html

4. Open the file and edit:
   
   Replace this:
   - [Article Title] ‚Üí "International Trade Theory"
   - [Category] ‚Üí "Macroeconomics"
   - [Date] ‚Üí "Feb 15, 2026"
   - [Read Time] ‚Üí "7 min read"
   - [Article excerpt...] ‚Üí Your summary
   - [Your article content...] ‚Üí Your full content

================================================================================
                     STEP 2: ADD TO ARTICLES DATABASE
================================================================================

1. Open 'data/articles.json'

2. Find the "articles" array (at the top)

3. Add your article entry:

{
  "id": "international-trade-theory",
  "title": "International Trade Theory",
  "description": "Exploring comparative advantage and trade benefits",
  "category": "Macroeconomics",
  "date": "2026-02-15",
  "featured": false,
  "tags": ["trade", "comparative advantage", "globalization"],
  "author": "Economics Notes",
  "readTime": "7 min read",
  "content": "articles/international-trade-theory.html",
  "gradient": "from-green-500 to-teal-600"
}

IMPORTANT:
- Add a comma AFTER the previous article
- Make sure "id" matches your HTML filename
- Use quotes around all strings
- Date format: YYYY-MM-DD

================================================================================
                       STEP 3: UPDATE CATEGORY COUNT
================================================================================

Still in 'articles.json', scroll down to "categories" section.

Find your article's category and increase the count:

{
  "name": "Macroeconomics",
  "slug": "macroeconomics",
  "count": 5     ‚Üê Increase this by 1
}

================================================================================
                           STEP 4: REFRESH WEBSITE
================================================================================

That's it! Your article will now appear on:
‚úÖ Homepage (in Recent Notes)
‚úÖ All Notes page
‚úÖ Categories page

If you set "featured": true, it will appear as the featured article!

================================================================================
                          üö® COMMON MISTAKES
================================================================================

‚ùå Forgetting comma after previous article
   ‚Üí JSON will be invalid!

‚ùå Mismatched article ID and filename
   ‚Üí Article won't open!

‚ùå Wrong date format (e.g., "Feb 15, 2026")
   ‚Üí Use: "2026-02-15"

‚ùå Category doesn't match existing categories
   ‚Üí Article won't filter correctly

‚ùå Not updating category count
   ‚Üí Count will be wrong on Categories page

‚ùå Missing quotes around strings
   ‚Üí JSON will be invalid!

================================================================================
                           üìû NEED HELP?
================================================================================

Article not showing up?

1. Check browser console (F12) for errors
2. Validate your JSON at jsonlint.com
3. Make sure article ID matches filename
4. Clear browser cache (Ctrl+Shift+R)
5. Check the article HTML file exists

JSON syntax error?

- Missing comma?
- Extra comma at end of array?
- Missing quote?
- Missing bracket?

Use jsonlint.com - it shows EXACTLY where the error is!
