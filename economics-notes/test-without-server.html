To see articles on the Home, All Notes, and Categories pages, you MUST use a local server!

The site uses JavaScript to load articles from data/articles.json, which browsers block when opening files directly (CORS policy).

Quick Start:

  1. Open Terminal/Command Prompt
  2. Navigate to the economics-notes folder
  3. Run: python -m http.server 8000
  4. Open browser to: http://localhost:8000
