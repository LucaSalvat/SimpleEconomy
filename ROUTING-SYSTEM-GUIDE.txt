================================================================================
                     üîÑ HOW IT WORKS
================================================================================

NAVIGATION FLOW:
1. User clicks internal link
2. navigation-manager.js intercepts click
3. Prevents default page reload
4. Adds fade-out effect
5. Updates browser history
6. Navigates to new page
7. Adds fade-in effect
8. Updates active navigation

PAGINATION FLOW:
1. Article page loads
2. article-pagination.js detects it's an article
3. Loads articles.json
4. Finds current article
5. Determines prev/next in category
6. Renders pagination UI
7. Renders related articles

BACK BUTTON FLOW:
1. User clicks browser back
2. Browser fires 'popstate' event
3. navigation-manager.js catches it
4. Smoothly transitions to previous page
5. No full page reload

                     üéØ ADDING PAGINATION TO ARTICLES
================================================================================

To add pagination to an article page, include these containers:

BEFORE CLOSING </main>:
```html
<div id="articlePagination"></div>
<div id="relatedArticles"></div>
```

BEFORE CLOSING </body>:
```html
<script src="../../js/article-pagination.js"></script>
```

FULL EXAMPLE:
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Article Title | Economics Notes</title>
    <link rel="stylesheet" href="../../config/styles.css">
</head>
<body>
    <header class="header">
        <!-- Header content -->
    </header>

    <main class="container">
        <!-- Article content here -->
        
        <!-- Pagination -->
        <div id="articlePagination"></div>
        
        <!-- Related Articles -->
        <div id="relatedArticles"></div>
    </main>

    <footer class="footer">
        <!-- Footer content -->
    </footer>

    <script src="../../js/navigation-manager.js"></script>
    <script src="../../js/article-pagination.js"></script>
</body>
</html>
```

The JavaScript automatically:
  ‚úÖ Detects the current article
  ‚úÖ Finds previous/next articles
  ‚úÖ Renders pagination controls
  ‚úÖ Shows related articles
  ‚úÖ Enables keyboard navigation
